## 🛠️ Technical Stack
I built this portfolio using modern web technologies to ensure it is both performant and visually engaging:

*   **Animations:** Three.js (for immersive 3D background elements)
*   **Typography:** Google Fonts (Inter & Outfit)
*   **Deployment:** GitHub Pages
